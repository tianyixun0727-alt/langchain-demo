# 7. RAG — Retrieval-Augmented Generation

Large Language Models have two key limitations: finite context windows and static training data. RAG addresses these by fetching relevant external knowledge at query time.

| File | Description |
|------|-------------|
| `01_overview.py` | Three RAG architectures with comparison |
| `02_agentic_rag.py` | Run an agent that decides WHEN to retrieve |
| `03_knowledge_base.py` | Build a searchable knowledge base from documents |
